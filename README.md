💣 Dynamite 66 (D66)

Write simple English → Build apps & games 🚀

Dynamite 66 (D66) is a beginner-friendly programming language where you can create apps, UI, and games using simple English-like commands.

---

🌐 Try Online

👉 https://yourname.github.io/dynamite-66/

---

⚡ Features

- 🧠 Easy English-like syntax
- 🎮 Game development (player + movement)
- 📱 Mobile-friendly controls
- 🔘 UI Buttons (MyButton system)
- 🖼 Image upload & display
- 🔊 Sound play support
- 🔁 Loops, conditions, functions
- ➕ Math & variables

---

🧪 Example Code

🎮 Game Example

make player at 50 80

speed player 5

🧠 Logic Example
